C++ Foundations

Bienvenido al curso C++ Foundations.

Este repositorio es un entorno educativo diseñado para enseñar C++ desde fundamentos reales, combinando:

  • Lenguaje C++ moderno
  • Control de versiones con Git
  • Compilación profesional con CMake
  • Pruebas automatizadas
  • Documentación estructurada con Doxygen

El objetivo del curso no es solo aprender sintaxis, sino aprender a trabajar como ingeniero de software.

Cómo usar este repositorio

Este repositorio forma parte activa del curso.

Reglas fundamentales:

  • El estudiante solo debe modificar la carpeta student/
  • Todo el contenido teórico vive en la documentación (Doxygen)
  • El flujo de trabajo con Git es parte de la evaluación

El sistema de CI del repositorio valida automáticamente:

  • Que solo se modifique student/
  • Que el proyecto compile correctamente
  • Que las pruebas (si existen) pasen

Navegación de la documentación

  • Main Page (esta página): visión general del curso

Estructura del repositorio

La estructura principal del repositorio es:

cpp-foundations/
├── student/
│ ├── include/
│ └── src/
├── docs/
│ ├── topics/
│ └── mainpage.dox
├── CMakeLists.txt
└── README.md

Significado de las carpetas:

  • student/: Código del alumno. ÚNICO lugar donde se debe programar.
  • docs/: Documentación del curso generada con Doxygen.
  • CMakeLists.txt: Sistema de compilación del proyecto.
  • README.md: Información general del repositorio para GitHub (no es la Main Page).

Flujo de trabajo esperado

El flujo de trabajo estándar del curso es:

  1. Leer la lección correspondiente
  2. Implementar código en student/
  3. Compilar y probar localmente
  4. Crear commits pequeños y claros
  5. Subir cambios a GitHub
  6. Verificar que el CI pase correctamente

No seguir este flujo suele resultar en errores, reprocesos o fallas de evaluación.

Compilación y pruebas locales

El proyecto se compila usando CMake.

Flujo mínimo recomendado:

mkdir -p build
cmake -S . -B build
cmake --build build
ctest --test-dir build --output-on-failure

El estudiante debe ser capaz de ejecutar estos comandos sin errores antes de avanzar en el curso.

Expectativas del curso

Se espera que el estudiante:

  • Lea y entienda las lecciones antes de programar
  • No copie código sin comprenderlo
  • Use Git de forma consciente
  • Aprenda a interpretar errores de compilación
  • Priorice claridad y corrección sobre rapidez

Este curso evalúa fundamentos, no atajos.

Nota final

C++ es un lenguaje poderoso, pero exige disciplina.

Este curso está diseñado para construir bases sólidas que permitan escalar a proyectos reales de ingeniería.

Comienza con la Lesson 00 antes de escribir una sola línea de código.
